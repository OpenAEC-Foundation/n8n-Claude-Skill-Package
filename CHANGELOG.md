## [1.0.0] - 2026-03-19

### Added
- 21 deterministic skills for n8n v1.x workflow automation
- **n8n-core/** (2 skills): architecture, REST API
- **n8n-syntax/** (8 skills): workflow JSON, expressions, extension methods, node types, credentials, code node, trigger nodes, AI nodes
- **n8n-impl/** (6 skills): custom nodes, deployment, workflow design, integrations, webhooks, security
- **n8n-errors/** (3 skills): execution, expressions, connection
- **n8n-agents/** (2 skills): review validator, project scaffolder
- Deep research vooronderzoek (20 sections covering entire n8n platform)
- Definitive masterplan with 7 batches and complete agent prompts
- INDEX.md with complete skill catalog
- README.md with installation instructions and skill table
- Core documentation (ROADMAP, DECISIONS, REQUIREMENTS, SOURCES, WAY_OF_WORK, LESSONS)

### Methodology
- 7-phase research-first development
- 3 parallel research agents for deep vooronderzoek
- 7 batches of 3 skill creation agents
- All code verified against official n8n documentation and source code
